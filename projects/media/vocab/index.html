Vocabulary

Media Server: This is the computer that hosts your media and distributes it to your various devices.

Media Center: This is the computer that connects to your TV and plays your media.

Kodi and Plex: A piece of software run on your media center computer. It provides you will an easy to use interface for interact with your media. It's optimized for a remote controller type device.

RAID: Software design to backup data across multiple drives. In the event of a drive failure, RAID is capable of swaping in a new drive, and rebuilding your data on it.

Headless Server: A server that does not have a Graphical User Interface (GUI). Instead, you interact with the server via a command line interface (CLI)

Graphical User Interface (GUI): The GUI is what you interact with on most computers and what you typcally associate with being your computer. Some examples include Gnome, KDE, and Unity

Command Line Interface: The CLI is a text based interface used to interact with computers. They are most common on headless servers. though every Linux system is capable of bringing up a CLI
